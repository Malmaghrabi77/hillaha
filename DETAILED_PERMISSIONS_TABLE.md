## ✨ الحالات الخاصة

### 1. Partner يرى بيانات فرعه فقط
```sql
WHERE partner_id = auth.user.partner_id
```

### 2. Store Admin يرى فرعه فقط
```sql
WHERE store_id = auth.user.store_id
```

### 3. Regional Manager يرى تحت إشرافه
```sql
WHERE invited_by = auth.uid()
   OR partner IN (his_partners)
```

### 4. Driver يرى طلباته فقط
```sql
WHERE driver_id = auth.uid()
```

### 5. Customer يرى طلباته فقط
```sql
WHERE customer_id = auth.uid()
```
