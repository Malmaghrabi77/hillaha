# حلّها — تطبيق العميل | Customer App

تطبيق الهاتف الذكي الأساسي لعملاء منصة حلّها لطلب الخدمات المنزلية والتوصيل.

## الميزات الرئيسية

- 🏠 **متجر الخدمات** — تصفح جميع الخدمات المتاحة
- 🔍 **البحث الحي** — ابحث عن الخدمات والمتاجر
- 🛒 **سلة الشراء والدفع** — شراء آمن مع التحويل البنكي
- 📦 **تتبع الطلبات** — تابع طلبك بالخريطة الحية
- 💳 **نقاط الولاء** — اجمع النقاط واستبدلها بمكافآت
- 🧹 **خدمات منزلية** — طلب تنظيف وكهرباء مباشرة

## البنية

```
apps/customer/
├── app/
│   ├── (tabs)/           # الشاشات الرئيسية
│   │   ├── home.tsx
│   │   ├── search.tsx    # بحث Supabase حي
│   │   └── orders.tsx
│   ├── services/         # خدمات منزلية
│   │   ├── cleaning.tsx  # حفظ Supabase
│   │   ├── electrical.tsx
│   │   └── delivery.tsx
│   ├── checkout.tsx      # دفع + رفع الإثبات
│   ├── loyalty.tsx       # نقاط من Supabase
│   └── tracking/
│       └── [orderId].tsx # تتبع حي
```

## مصادر Supabase

- `partners` — المتاجر والخدمات
- `orders` — قائمة الطلبات
- `service_bookings` — حجوزات الخدمات المنزلية
- `delivery_requests` — طلبات التوصيل
- `loyalty_points` — نقاط الولاء للعميل
- `payment_proofs` — إثباتات الدفع (Storage)

## التشغيل

pnpm dev:customer
