# حلّها — تطبيق المندوب | Driver App

تطبيق الهاتف الذكي لمندوبي التوصيل على منصة حلّها.

## الميزات الرئيسية

- 📦 **طلبات التوصيل الحية** — استقبل طلبات جديدة
- 🗺️ **التتبع بـ GPS** — تتبع موقعك الحي أثناء التوصيل
- 🚴 **إدارة حالة الطلب** — تحديث الحالة (التقاط ← الطريق ← تسليم)
- 💰 **الأرباح المباشرة** — عرض الأرباح اليومية والشهرية
- 📊 **إحصائيات الأداء** — عدد التوصيلات والتقييم
- 👤 **حسابك الشخصي** — بيانات المندوب والحساب البنكي

## البنية

```
apps/driver/
├── app/
│   ├── (tabs)/           # الشاشات الرئيسية
│   │   ├── home.tsx      # الطلبات المتاحة
│   │   ├── active.tsx    # التوصيل النشط مع GPS
│   │   ├── earnings.tsx  # الأرباح والإحصائيات
│   │   └── profile.tsx   # الملف الشخصي
│   ├── (auth)/
│   │   ├── login.tsx     # تسجيل دخول + بصمة
│   │   └── register.tsx  # التسجيل مع صورة
```

## مصادر Supabase

- `orders` — الطلبات (status, driver_id, lat/lng)
- `profiles` — بيانات المندوب

## الميزات المتقدمة

- ✅ بصمة + كلمة مرور
- 📍 تحديث الموقع كل 4 ثواني أثناء التوصيل
- 🗺️ خرائط Google Maps (تتبع مباشر)
- 🔔 إشعارات الطلبات الحية

## التشغيل

```bash
pnpm dev:driver
```
