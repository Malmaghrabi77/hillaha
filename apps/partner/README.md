# حلّها — لوحة تحكم الشريك | Partner Dashboard

موقع ويب لإدارة متجر الخدمات والطلبات الخاصة بك على منصة حلّها.

## الميزات الرئيسية

- 📊 **لوحة المعلومات** — إحصائيات الطلبات والأرباح
- 🏪 **إدارة المتجر** — تحديث معلومات العمل والساعات
- 📋 **قائمة الطلبات** — عرض وتحديث حالة الطلبات
- 👥 **إدارة فريقك** — عرض مندوبيك والتقييمات
- 📈 **التقارير** — تحليلات شاملة للأداء

## البنية

```
apps/partner/
├── app/
│   ├── dashboard/       # لوحة المعلومات الرئيسية
│   ├── menu/           # إدارة قائمة الخدمات
│   │   ├── services.tsx
│   │   └── pricing.tsx
│   ├── orders/         # إدارة الطلبات
│   └── settings/       # إعدادات المتجر
├── components/         # مكونات React
├── lib/               # دوال مساعدة
└── public/            # أصول ثابتة
```

## مصادر Supabase

- `partners` — بيانات المتجر
- `services` — الخدمات المتاحة
- `orders` — الطلبات الواردة
- `partner_logos` — شعار المتجر (Storage)
