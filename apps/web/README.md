# حلّها — الموقع الإعلامي | Marketing Website

الموقع الرسمي للتعريف بمنصة حلّها والتسويق لخدماتنا.

## الغرض

- 🌐 صفحة هبوط احترافية
- 📱 عرض الخدمات المتاحة
- 💼 معلومات الشركة
- 📞 نموذج التواصل
- 📊 عرض الإحصائيات

## التكنولوجيا

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS (أو styled-components)

## التشغيل

```bash
pnpm dev:web
```

## البناء

```bash
pnpm --filter hillaha-web build
```

## الانتشار

يتم الانتشار تلقائياً على Vercel عند الدفع إلى فرع `main`.
